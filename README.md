# Dash

> An experimental shell.

## Features

- [x] Exit command
- [x] Background processes(`&`).
- [x] cd command
- [x] `;` implementation
- [x] `&&` implmentation
- [x] line continuation using `\`
- [ ] Pipes
- [ ] Redirection
- [ ] Jobs and signal forwarding.
- [ ] Custom parser
- [ ] GNU readline for command line input.
- [ ] In command substitutions and expansion.
